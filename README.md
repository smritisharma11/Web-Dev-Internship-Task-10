# Task 10: Consume a Public API Using Fetch API

## Project Description
In this task, I developed an API-driven web page that fetches and displays data from an external public source. This project explores asynchronous JavaScript, network handling, and dynamic UI updates using the Fetch API.

## Key Features Implemented
* **Asynchronous Data Fetching**: Used the `fetch()` API with `async/await` to send HTTP requests to a public endpoint.
* **JSON Handling**: Successfully parsed API responses into JSON format to extract and use specific data fields.
* **Dynamic DOM Rendering**: Updated the UI in real-time to display the fetched content without reloading the page.
* **Error & Loading States**: Implemented a loading indicator and comprehensive error handling for network failures or invalid responses.
* **Code Documentation**: Added detailed comments to explain asynchronous logic and flow.
